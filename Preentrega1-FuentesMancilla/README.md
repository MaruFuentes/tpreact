# E-COMMERCE 4 ESQUINAS 3D

  

App de ventas en linea de productos impresos en 3D, la app se encarga de mostrar todos los productos también están separados por categorías, así como también detalle por producto, carrito de compras donde se almacenan momentáneamente todos los productos seleccionados, y un formulario para completar la compra


  

## version actual 1.0.5


  ## Librerias externas utilizadas
